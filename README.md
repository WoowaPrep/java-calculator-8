# 문자열 덧셈 계산기

## 프로젝트 개요

입력한 문자열에서 숫자를 추출해 덧셈을 수행하는 계산기입니다.
기분 구분자(쉼표, 콜론)와 사용자 정의 커스텀 구분자를 지원하며,
잘못된 입력에 대한 예외 처리를 포함합니다.

## 주요 기능

- 쉼포(,) 또는 콜론(:)을 구분자로 하는 숫자 덧셈
- 커스텀 구분자 지정 가능 (`//커스텀 구분자\n숫자` 형식)
- 음수 및 잘못된 입력에 대한 예외 처리
- 빈 문자열 입력 시 0 반환
- BigInteger를 활용한 큰 숫자 연산 지원

## 실행 예시

```
뎃셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
```

```
뎃셈할 문자열을 입력해 주세요.
//;\n1;2:3,4
결과 : 10
```
